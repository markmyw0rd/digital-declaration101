# AURTTE104 Digital Declaration (Vercel)

Stack: Next.js (App Router) + Tailwind + Vercel Postgres + Vercel Blob + Resend + pdf-lib.

## Env Vars
Set in Vercel → Settings → Environment Variables
```
POSTGRES_URL=...
POSTGRES_PRISMA_URL=...
POSTGRES_URL_NON_POOLING=...
POSTGRES_USER=...
POSTGRES_HOST=...
POSTGRES_PASSWORD=...
POSTGRES_DATABASE=...
BLOB_READ_WRITE_TOKEN=...
RESEND_API_KEY=...
APP_URL=https://<your-vercel-domain>
JWT_SECRET=<64-byte-random>
```

## First run
1. `npm i`
2. `npm run db:push` (locally with `VERCEL_POSTGRES_*` envs set or use SQL console online)
3. `npm run dev`
